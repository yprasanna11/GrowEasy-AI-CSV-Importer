# GrowEasy AI CSV Importer 🚀

An AI-powered CRM CSV data importer that allows users to upload CSV files and process customer data using AI.

The application extracts CSV data, processes the information, and provides structured insights using AI-powered analysis.

## ✨ Features

* 📄 Upload CRM CSV files
* 🔍 Parse and extract CSV data
* 🤖 AI-powered data processing
* 📊 Generate structured CRM insights
* ⚡ Fast API-based processing
* 🌐 Modern responsive web interface

## ⚙️ Installation and Setup

### Clone Repository

```bash
git clone https://github.com/yprasanna11/GrowEasy-AI-CSV-Importer.git
```

Move into project directory:

```bash
cd GrowEasy-AI-CSV-Importer
```

## Backend Setup

Navigate to backend:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Start backend server:

```bash
npm run dev
```

Backend runs on:

```
http://localhost:5000
```

## Frontend Setup

Open another terminal.

Navigate to frontend:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
npm run dev
```

Frontend runs on:

```
http://localhost:3000
```

## 📤 CSV Upload Flow

1. User selects a CRM CSV file.
2. Frontend sends the file to backend API.
3. Backend parses CSV data.
4. AI service processes the information.
5. Structured output is returned to the user.
